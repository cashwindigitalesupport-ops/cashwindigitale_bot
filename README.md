## 📝 Description

Bienvenue sur le bot Cashwin Digitale! Cette application vous permet de gagner de l'argent facilement et rapidement.

## 🚀 Démarrage

Pour commencer, veuillez suivre les instructions d'installation ci-dessous.

### Installation

```bash
# Clonez le dépôt
git clone https://github.com/cashwindigitalesupport-ops/cashwindigitale_bot.git

# Installez les dépendances
npm install
```

### Configuration

Configurez vos paramètres dans le fichier `.env`

### Utilisation

```bash
npm start
```
